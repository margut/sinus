# Sinus Regal
Webanwendung für eine parameterisierbare Bauanleitung.

![Beispielbild](/img/IMG_1180.jpeg)

Änderbar sind
* Alle Maße
* Farben
* Anzahl Fächer

## Ausgabe
* PDF-Datei
  - [x] Stückliste der Bauteile mit Maße
  - [x] Größe der Flächen der unterschieldlichen Farben.
* Bauanleitung
* Beispielbilder
